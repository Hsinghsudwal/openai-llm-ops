# openai-llm-ops


### How to run:

1. Create a new environment 

```bash
conda create -n llmops python=3.10 -y
```

2. Activate the environment

```bash
conda activate llmops
```

3. Install required packages

```bash
pip install -r requirements.txt
```


4. Run the app.py

```bash
streamlit run app.py
```
